1. node JS : JS이긴 하나, 통상적으로 브라우저에서 실해되는 JS와 다르게 브라우저 밖에서도 돌아갈 수 있음.
nodeJS 설치 후, 터미널에 node입력후 버젼 나오면 성공. -> npm 같이 설치가능

2. npm init을 통해 package.json 생성

3. .gitignore 통해 git할 필요없는 큰 파일목록들을 작성함(package-lock.json,npm-module같은 파일들..)

4. npm install express --save(package.json에 등록)

5. package.json 의 "scripts"{"start": "node index.js"}로 변경 

6. (babel사용시) npm install @babel/preset-env 후, .babelrc파일 생성후, (ES-6사용)

{
  "presets": ["@babel/preset-env"]
}

과 같이 입력 후 저장. 추가적으로 scripts 내용 역시 babel-node index.js로 변경.


7. 저장 시 마다 새로고침 되는 기능추가(nodemon) : npm install nodemon -D (프로젝트에 관련없는 기능추가임. "devdependencies")

8. "scripts":{ "start":"nodemon --exec babel-node index.js"}로 변경 후, 실행.

9. middleware 사용..항상 다른함수보다 위에 두어야 실행됨.

라우터와 실행함수 중간기능을 담당한다.
